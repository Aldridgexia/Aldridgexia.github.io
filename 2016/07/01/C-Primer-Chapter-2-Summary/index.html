类型转换

  • bool b = 42; //b 为真
    • 把非布尔类型的算数值赋给布尔类型时，初始值为0时结果为false，否则结果为true
  • int i = b; //i 的值为1
    • 把布尔值赋给非布尔类型时，初始值为false 则结果为0， 为true 时结果为1
  • i = 3.14； //i的值为3
    • 把浮点数赋给整数类型时，仅保留浮点数中小数点前的部分
  • double pi = i; //pi 的值为3.0
    • 把整数值赋给浮点类型时，小数部分为0，如果整数所占空间超过浮点类型的容量，精度可能有损失
  • unsigned char c = -1; //假设char 占8比特， c 的值为255
    • 赋给无符号类型一个超过它表示范围的值时，结果是初始值对无符号类型表示数值总数取模后的余数。
  • signed char c2 = 256; //假设char 占8比特，c2 的值是未定义的
    • 赋给带符号类型一个超过它表示范围的值时，结果是未定义的 (undefined)
